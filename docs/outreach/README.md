# Outreach Kit

Bu klasör QrMatik SEO backlink çalışmaları için temel varlıkları içerir.

## İçerikler
- `targets.csv`: İlk 10 hedef domain, iletişim epostası ve durum.
- `email-templates.md`: İlk iletişim, takip ve kapanış e-posta şablonları.
- `README.md`: Kullanım talimatı.

## Kullanım Akışı
1. `targets.csv` dosyasındaki `status` alanını güncelle (queued -> contacted -> replied -> published).
2. İlk şablonu gönder; 5 gün içinde yanıt yoksa follow-up; 10. günde kapanış.
3. Yayına alınan backlinkleri Search Console’da doğrula.
4. Ay sonunda yeni 10 domain ekle.

## Notlar
- Anchor çeşitliliğini koru (marka + kısmi eşleşme).
- Düşük kaliteli (spam/toksik) domainleri ekleme.
- Geri link karşılığı ücret teklif etme (Google yönergelerine aykırı).
